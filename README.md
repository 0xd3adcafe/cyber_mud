Cyberpunk text MUD set in **Night City** вҖ” jack in, take gigs, hack the grid, and survive the after-dark economy. Forked from the original **mud** project; **263 rooms**, **109 NPCs**, and a built-in **Textual TUI** so you play in-repo, not in a legacy MUD client.

**Default locale: English.** Use `lang zh` in-game for Traditional Chinese. See [docs/LOCALIZATION.md](docs/LOCALIZATION.md).

## Why jack in?

Original Night City fiction with **homage vibes** you will recognize вҖ” neon rain and corporate dystopia (**Blade Runner**), street cred, cyberware, and quickhacks (**Cyberpunk 2077**), plus a **Watch Dogs**-style surveillance-hacking roadmap (profiler, ctOS hacks, digital footprint). Not official IP; built to *feel* like the worlds you already love.

### Play the city

| | |
|---|---|
| **Explore** | Districts, rooftops, underground, transit, vehicles, housing & stash |
| **Fight** | Real-time combat, CP2077 weapon types, quickhacks, taunt/finish, corpses & loot |
| **Run the net** | NETRUN nodes, `hack` / `probe`, RAM, environment interaction mid-run |
| **Live the grind** | Gigs & journal, street cred, shops, craft, braindance, NCPD heat, faction AI |
| **Grow** | Levels, talents, nine cyberware slots, romance scaffolding, `sit` / `rest` / `sleep` vitals |

Tutorial yard onboarding вҶ’ Watson flats вҶ’ Kabuki clubs and corpo hubs. Type `look`, `scan`, `go`, `talk`, `gigs` вҖ” the city answers back.

### Optional 18+ (opt-in)

Mature content is **off by default** (`teen` rating). Register with `mature` or `settings mature on` to unlock 18+ venues, romance beats, mature gigs/braindances, and grittier combat copy вҖ” gated at login and command layer. Pack lives in private **`cyber_mud_mature`**; public clones stay teen-safe. See [docs/MATURE_CONTENT.md](docs/MATURE_CONTENT.md).

### Built for players and builders

- **Textual TUI** вҖ” sidebar PDA, live map, equipment, gigs tracker, Tab completion, `/theme` (incl. ctOS / DedSec / Profiler skins), auto-reconnect  
- **Data-driven world** вҖ” rooms, NPCs, quests, and locale in YAML; `./run.sh --dev` hot-reloads data and code  
- **Bilingual** вҖ” `en` default, `zh` opt-in; player guides under [docs/player/](docs/player/README.md)  
- **Agent-friendly repo** вҖ” [CLAUDE.md](CLAUDE.md), phased backlog [PHASES.md](docs/PHASES.md), `./admin.sh validate`

```bash
./setup.sh && ./run.sh          # server
./run.sh --client               # jack in вҖ” this is the real game
```

## Core principles

1. **English default locale (mandatory)** вҖ” `locale=en` at runtime; `lang zh` is opt-in. See [CLAUDE.md](CLAUDE.md) В§ Project rules and [docs/LOCALIZATION.md](docs/LOCALIZATION.md).  
2. **Built-in Textual client** вҖ” no third-party MUD clients for production play.  
3. **World in `data/`** вҖ” code interprets YAML, not hardcoded rooms.  
4. **One command per module** under `commands/`.  
5. **Bilingual mirrors** вҖ” `en.yaml` + `zh.yaml`, English `*.md` + `*.zh.md`.

## Quick start

First time? Read [Getting Started](docs/player/GETTING_STARTED.md) вҖ” register, `look`, leave the tutorial yard.

```bash
cd cyber_mud
./setup.sh          # first run: install deps
./run.sh            # start server
./run.sh --client   # another terminal: TUI client
./admin.sh validate # validate world data + tests
```
